# Variable Income budgeting application

Application description

Backend:  https://github.com/Michael-P-Drury/group-19-backend/tree/main

Frontend: https://github.com/Michael-P-Drury/group-19-frontend/tree/main

# Installation

Get the backend and frontend of the application

```bash
mkdir group-19-project
cd group-19-project
git clone https://github.com/Michael-P-Drury/group-19-backend.git
git clone https://github.com/Michael-P-Drury/group-19-frontend.git
```

# Setup
Download all the required node packages

```bash
npm install
```

# Usage
To run the frontend of the application
```bash
npm run dev
```
